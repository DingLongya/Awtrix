Privacy policy


What data is collected?

Your IP address to activate the premium functionality via premium key

What do I use your data for?

This IP Adress is imperatively necessary for the license server. The current IP Adress during activation process of your premium key is stored once in the Premium license Server to differ your AWTRIX and your Licence key from others. After successful activation, the IP address will not be collect in the future. No other personal information or preferences are collected, stored or shared.  

How i can prevent AWTRIX to use my data

If you deactivate premium from every AWTRIX instance, also your IP entry will be deleted from the license server.

AWTRIX CLOUD

I have developed the AWTRIX Cloud with security in mind. No further configuration is required to make the instance accessible to the Internet. Furthermore, no user names, passwords, e-mail addresses or any other private information are required and the cloud server does not store any information about the connections. Youre completely anonym.

Icons and Animations

Your icons and animations are uploaded completely anonymously.

AWTRIX Apps

All apps and their settings remain locally on your AWTRIX. No data is sent to the AWTRIX Cloud server. Registration data or API keys are used exclusively for the official API of the respective service. This privacy statement does not apply these third party services wich is used by AWTRIX. I cannot guarantee that these third parties handle your personal data in a reliable or secure manner. I recommend you read the privacy statements of these websites prior to making use of these websites.

General note

I would like to point out that data transmission on the Internet can have security gaps. A complete protection of data against access by third parties is not possible.
